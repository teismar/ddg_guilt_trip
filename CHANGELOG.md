# Changelog

## 0.1.0

- **Initial Release**: Core interception for `!g`, `!yt`, `!a`, `!r`, `!gh`, `!w`.
- **Strict Mode**: Added option to auto-redirect without showing the banner.
- **Privacy Stats**: Added "Big Tech Avoided" counter with 30-day history and detailed breakdown.
- **Snooze**: Added 30-minute pause functionality with countdown timer.
- **Custom Redirects**: Added inline editor to change destination URLs (e.g., custom Piped instances).
